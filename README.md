# Supernal Desktop — Feedback & Bug Reports

This is the public issue tracker for [Supernal Desktop](https://supernal.ai).

## Filing a bug

Use the in-app bug reporter (Help → Report a Bug) for the best experience — it auto-fills system info and anonymizes your data before posting.

Or [open an issue](https://github.com/supernalintelligence/supernal-desktop-feedback/issues/new/choose) directly.

## Privacy

Reports filed via the in-app reporter contain:
- Your description of the bug
- Page section (anonymized — no account names or IDs)
- Browser/OS info

They never contain your email address, account details, or any private data.

## Code

The Supernal Desktop source code is in a private repository. This repo is for issues only.
